# CodeAlpha Calculator

This is a basic calculator project created as part of the CodeAlpha Frontend Development Internship.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Delete button
- Keyboard support
- Responsive layout

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open `index.html` in a browser.
2. Click the calculator buttons or use your keyboard.
3. Press `Enter` for result.
4. Press `Escape` to clear the screen.
5. Press `Backspace` to delete the last character.
